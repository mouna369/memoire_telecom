2. Normalisation Arabe
- Supprimer Tashkeel (voyelles)  كِتَابٌ  →  كتاب
- Supprimer Tatweel              كـــتاب  →  كتاب
- Normaliser Hamza               أ إ آ    →  ا
- Normaliser Ta marbuta          مدرسة   →  مدرس
- Normaliser Alef maqsura        على     →  علي
- Normaliser Ya                  ي  →  ى

3. Normalisation Dialecte Algérien
- Normaliser les répétitions     واااااو  →  واو
- Unifier les variantes          
  هكذا / هكاك / هكا  →  هكذا
  واش / وش           →  ماذا
  راهو / راه         →  هو
- Gérer le code-switching        
  "réseau" + "شبكة"  →  garder les deux



  4. Suppression Stopwords
- Stopwords Arabe standard
  في، من، إلى، على، هذا، هذه...

- Stopwords Dialecte Algérien
  واش، راهو، كيما، بصح، حتى...

- Stopwords Français
  je, tu, il, le, la, les, de...

- Stopwords spécifiques Télécom
  salam, bonjour, bonsoir, svp, stp


  5. Tokenisation
- Découper le texte en tokens (mots)
  "خدمة الانترنت سيئة" → ["خدمة", "انترنت", "سيئة"]

- Gérer les mots composés
  "ما يخدمش" → ["ما", "يخدم", "ش"]  
               ou garder → ["مايخدمش"]

- Filtrer les mots courts (< 2 lettres)
  و، أ، ب، ك  → supprimer
